## Publish To GitHub Later

1. GitHub 계정을 만든다.
2. 새 저장소를 만든다. 예: `jhlee-portfolio`
3. 이 폴더의 파일을 저장소에 업로드한다.
4. 저장소 설정에서 GitHub Pages를 켠다.
5. `main` 브랜치 / root 기준으로 배포한다.
